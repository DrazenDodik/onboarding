# Valohai Onboarding Sample :shark::rocket:

## Prerequisites
* Python 3 and pip

## Setup
* `pip install valohai-cli`
* `vh login`
* (optional) Install the Jupyter notebook addon
    * `pip install notebook`
    * `pip install jupyhai`
    * `jupyhai install`
